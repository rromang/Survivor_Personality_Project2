Summary: Latitude vs. X

The purpose of this project was to analyze how weather changes the closer you get to the equator. To accomplish this analysis, we first pulled data from the OpenWeatherMap API to assemble a dataset on over 500 cities.

After assembling the dataset, we used Matplotlib to plot various aspects of the weather versus city latitude. Weather factors analyzed: max temperature, humidity, cloudiness, and wind speed. This site provides the source data and visualizations created as part of the analysis, as well as any trends and correlations observed.
